# 三、死亡惩罚与装备快照（投保）

- **死亡时**：身上**全部装备与物品**消失，包含**口袋、背心、背包、载具**内一切物品；**拖拽态载具**及其内物品**一律消失**（不留在世界）。实现时须**清空绑定载具引用**（如 `bound_vehicle_id`）与 **inventory_vehicle**，避免复活后仍引用已消失载具。战斗技能扣 50 级（不扣到 0 以下）；传送到“地狱”对话后回基地；基地仓库与生产存货不受影响。**死亡后不得重新发放新手套装**：新手套装（default_equipment）仅用于**新游戏**初始化角色时穿戴，死亡复活后装备栏为空，不自动补发任何默认装备。
- **复活后**：**心情回到正常程度**（如恢复至心情中心值）；**体力、底气、饱食、饮水等恢复至一半**（即恢复至各自上限的 50%，或实现约定等效规则）。**肢体重置为完好**（功能恢复），但**损毁值只恢复至一半**（各部位损毁值 = 该部位损毁值上限的 50%，或实现约定采用“当前值减半”等规则，以策划确认为准）。**四类物品栏**（口袋、背心、背包、载具）均为空，不保留任何物品。
- **投保**：下地牢前可付费对当前穿戴装备做一次快照；同一时间仅 1 个生效快照。
- **代码**：死亡后 NPC 给出一段 AES-256 加密代码（含玩家信息、地牢标识、快照装备等），玩家不可反推内容。
- **使用代码**：必须在同一地牢同一入口输入；进入后地牢仍完全随机，系统随机指定一房间生成可互动元素；需到达该房间并成功撤离才可从 NPC 取回装备（仅装备，消耗品不恢复）。
- **消耗**：一旦使用代码，不论成败，该玩家此代码即消耗；未到指定房间或死亡则失去机会。
- **共享**：同一代码可被多玩家使用；每人仅一次机会；成功则各自从 NPC 获得那套装备。
