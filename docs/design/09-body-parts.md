## 八、身体部位与状态效果

- **部位**：头、胸、腹、左手、右手、左脚、右脚；命中部位按权重随机（躯干高、头低、四肢中等）。
- **效果**：
  - **头**：本回合眩晕无法行动；会自己恢复。
  - **胸**：只算伤害。
  - **腹**：一次性降低饱食度与饮水度。
  - **手**：命中/招架下降会自己恢复；有几率骨折，骨折与脚同，须特殊手段解除。
  - **脚**：暂时无法移动会自己恢复；有几率骨折 → 脚部技能失效、移动一格需两回合，须特殊手段解除。
- **肢体损毁**：每个部位有**损毁值**；损毁值满后该部位标记为**损毁**、失去功能（四肢还包括无法出招等，见技能篇），消耗品无法降低；可通过**手术用品**暂时扣除部分损毁值上限以恢复功能，**手术**技能等级影响可恢复的损毁值上限比例。
- **已损毁部位再受击时的损毁重分配**：
  - 命中某一部位且照常走完减伤链后，得到该次**最终伤害**，并按既有规则向下取整得到本次**损毁增加值** \(Q\)（与 5.6.3 一致，与用于扣血的数值同源）。
  - 若**命中部位当前尚未损毁**：\(Q\) **全部**加在该部位损毁值上（仍受该部位损毁上限封顶，超出部分作废）。
  - 若**命中部位已经损毁**：**不得**再把 \(Q\) 加在该部位；将 **\(Q\) 整体**分摊到当前**尚未损毁**的全部部位上（七部位里损毁值未满、未标记损毁者）。**分摊方式**：设未损毁部位数为 \(n\)；每部位先加 \(\lfloor Q/n \rfloor\)，余数 \(R = Q \bmod n\) 按固定顺序 **头 → 胸 → 腹 → 左手 → 右手 → 左脚 → 右脚** 只计入「未损毁」部位，从顺序上最靠前者开始，依次多 **+1** 共 \(R\) 次，保证各部位本次增量之和恰为 \(Q\)。每部位加完后照常判定是否顶满损毁。
  - **扣血**：仍按本次**最终伤害**与命中部位常规结算（不因目标部位已损毁而改为 0 伤害），除非别处另有规则。
  - **全无未损毁部位**：若 \(n = 0\)（七部位均已损毁），则本次 **\(Q\) 作废**，不累加到任何部位。
  - 敌人与玩家共用本规则（七部位与损毁逻辑一致，见敌人篇）。
- **各部位损毁值上限**：

  | 部位 | 损毁值上限 | 说明 |
  |------|-----------|------|
  | **头** | 50 | 头部耐受度最低，少量损毁积累即可导致功能丧失。 |
  | **胸** | 100 | 标准上限，有肌肉与肋骨支撑，较能承受累积伤害。 |
  | **腹** | 80 | 软组织与空腔器官集中，耐受度低于胸部。 |
  | **左手/右手** | 100 | 标准上限；骨折等会更早触发，但损毁积累速度较快。 |
  | **左脚/右脚** | 100 | 标准上限；骨折与行动失能早于损毁满值触发。 |
